# Projeto 4 - Augmented Analytics com Inteligência Artificial e LLMs

# Instruções:

# 1- Baixe o arquivo zip final do capítulo, descompacte e coloque em uma pasta no seu computador (sem espaço e sem acento no nome da pasta).

# 2- Usando um editor de texto crie um arquivo .env e adicione a linha abaixo incluindo sua chave API da OpenAI conforme instruções das aulas:

OPENAI_API_KEY=your_openai_api_key

# 3- Abra o terminal ou prompt de comando, navegue até a pasta com os arquivos e ative um ambiente virtual com os comandos abaixo:

python -m venv dsavenv
source dsavenv/bin/activate

# Obs: No Windows use `dsavenv\Scripts\activate`

# 4- Instale os requerimentos:

pip install -r requirements.txt

# 5- Execute a app:

python app.py

# 6- Abra o navegador e acesse http://127.0.0.1:5000

# 7- Faça perguntas relacionadas a base de dados usada pelo LLM:

O que a pesquisa recente da Salesforce descobriu?

Qual percentual de empregadores estão tendo problemas para preencher vagas digitais com candidatos qualificados?

Qual a habilidade mais importante na era da Inteligência Artificial?

# 8- Quando concluir, desative o ambiente virtual:

deactivate
